# CFA Epinow2 Pipeline

This CFA project is a lightweight wrapper around [{EpiNow2}](https://github.com/epiforecasts/EpiNow2), which adds functionality for deployment in Azure Batch. `{EpiNow2}` estimates the time-varying reproduction number, growth rate, and doubling time using a range of open-source tools ([Abbott et
al.](https://doi.org/10.12688/wellcomeopenres.16006.1)), and current
best practices ([Gostic et al.](https://doi.org/10.1371/journal.pcbi.1008409)).  
The `EpiNow2 Pipeline` includes helper functions to:

* Interface with Azure services
* Convert input data to the expected `{EpiNow2}` input format
* Save outputs in standardized formats
* Add logging and metadata 

Click [here](https://github.com/CDCgov/cfa-epinow2-pipeline/tree/main) to go to the GitHub repository.
